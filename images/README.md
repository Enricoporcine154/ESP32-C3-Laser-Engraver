## 📐 图片规范

### 尺寸要求
- **产品照片**: 推荐 1920×1080 或更高
- **截图**: 推荐 1280×720 或更高
- **GIF**: 推荐 800×600，< 5MB

### 格式要求
- **照片**: JPG/JPEG（压缩质量 85%）
- **截图**: PNG（无损）
- **GIF**: 优化后（使用 [ezgif.com](https://ezgif.com/optimize)）

### 命名规范
- 使用小写字母
- 使用连字符分隔单词
- 避免空格和特殊字符

---

## 📸 拍摄建议

### 产品照片
1. **光线**: 使用自然光或柔光箱
2. **背景**: 使用纯色背景（白/黑/灰）
3. **角度**: 正面、侧面、45度角
4. **清晰度**: 确保对焦准确

### 工作演示
1. **激光防护**: 佩戴激光防护眼镜
2. **环境**: 保持整洁的工作台
3. **光线**: 拍摄雕刻过程时可调暗环境光
4. **安全**: 确保没有易燃物
