# 硬件设计文件

本目录包含 ESP32-C3 激光雕刻机的硬件设计文件。

---

## 📂 目录结构

```
hardware/
├── schematic.pdf        # 电路原理图
├── pcb_layout.png       # PCB 布局图
├── bom.csv             # 物料清单 (可选)

    ├── pcb_front.jpg   # PCB 正面
    ├── pcb_back.jpg    # PCB 背面
    └── assembled.jpg   # 组装完成照
```

## 🔧 硬件规格

### 核心芯片
- **MCU**: ESP32-C3FH4
  - 内置 4MB Flash
  - RISC-V 单核 160MHz
  - 支持 WiFi + BLE 5.0

### 电机驱动
- **型号**: A4988 步进驱动模块 × 2
- **细分**: 16 细分（MS1/MS2/MS3 全接高电平）
- **电流**: 推荐 0.3-0.4A（光驱电机）

### 电源
- **输入**: 12V 2A DC
- **方案**: USB PD 诱骗模块（可选）

### 激光模块
- **功率**: 250-500mW
- **控制**: 5V PWM (TTL 电平)
- **波长**: 450nm（蓝光）

## 📝 引脚定义

| 功能 | GPIO | 说明 |
|------|------|------|
| X_STEP | GPIO0 | X 轴步进脉冲 |
| X_DIR | GPIO4 | X 轴方向 |
| Y_STEP | GPIO1 | Y 轴步进脉冲 |
| Y_DIR | GPIO5 | Y 轴方向 |
| ENABLE | GPIO10 | 电机使能（低有效） |
| LASER_PWM | GPIO3 | 激光 PWM 控制 |
| LED | GPIO8 | 板载 LED（可选） |

---

## 🔌 接口说明

### 电机接口
- **X 轴**: 4Pin 排针（STEP, DIR, GND, VMOT）
- **Y 轴**: 4Pin 排针（STEP, DIR, GND, VMOT）
- **共使能**: EN 引脚并联控制

### 激光接口
- **3Pin 排针**: PWM, VCC(5V), GND

### 电源接口
- **DC 插座**: 5.5×2.1mm（12V 输入）
- **USB-C**: 供电 + 固件烧录

---

## 🛠️ PCB 打样建议

### 推荐参数
- **层数**: 2 层
- **板厚**: 1.6mm
- **铜厚**: 1oz (35μm)
- **表面处理**: HASL（有铅喷锡）
- **最小线宽/间距**: 6/6mil
- **最小孔径**: 0.3mm

### 成本估算（嘉立创/捷配）
- **5 片 10×10cm**: 约 ¥10-20
- **交期**: 24 小时

---

## ⚠️ 安全警告

1. **电源安全**: 确保 12V 电源极性正确，反接会烧毁芯片
2. **激光安全**: 上电前确保激光模块已正确连接，避免直视光束
3. **电机安全**: 通电状态下禁止拔插电机，会损坏驱动芯片
